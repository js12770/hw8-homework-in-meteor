### My Homework

这是一个基于meteor与nodejs的在线作业发布与提交网站。

### Install && Run

1. 这是在Ubuntu 14.04 下开发的，所以在Windows下的兼容性并不清楚...
2. 你需要安装NodeJs与Meteor
3. 进入文件根目录，然后命令行输入`meteor run`
4. 访问`http://localhost:3000`即可使用
